# MASHUP
## Overview
A Python-based tool to create seamless mashups of multiple songs. This project automatically blends tracks, synchronizes beats, and generates a single combined audio output.
## Feature
- Combine multiple songs into one mashup.
- Automatic beat synchronization for smooth transitions.
- Supports common audio formats like MP3 and WAV.
- Lightweight and easy-to-use Python script.
## Technologies Used
- pydub – For audio manipulation
- FFmpeg – Required backend for audio processing
